# Rhodes

Web-based Fender Rhodes electric piano emulator.

**[rhodes.nicoacosta.xyz](https://rhodes.nicoacosta.xyz)**

## Features

- Sampled Rhodes electric piano tones with 12-voice polyphony
- 2-octave interactive keyboard with octave shifting
- Effects chain: EQ (Volume/Tone), Tremolo (Rate/Depth), Chorus (Mix)
- Play with mouse, touch, or computer keyboard (24 mapped keys)
- Vintage hardware-accurate UI with SVG rotary knobs and chrome trim

## Tech stack

- Next.js 16, React 19, TypeScript, Tailwind CSS 4
- Web Audio API for synthesis and effects
- Vitest for testing
